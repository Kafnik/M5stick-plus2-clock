## Что это такое ? 
Это скрипты для M5 stick plus2 внем сделанны часы и календарь когда вы запустите время будет неправильным вам нужно нажать кнопку маленькую это не кнопка M5 и не кнопка питания.

## Как управлять 

1. Кнопка M5 сделанна для прибавления на +1
2. Кнопка боковая это не питания кнопка отвечает на пролистование
3. Кнопка питания она отвечает за сохранение всех извенений

Я назвал систему M5STACK OS
